# Tab + Vim 快捷键扩展方案
# 2026-02-02

================================================================================
一、当前已有的 Tab 功能
================================================================================

Tab + h/j/k/l  → 方向键（上下左右）
Tab + 1~0/-/=  → F1~F12
Tab + 退格     → 向前删除 (Delete Forward)


================================================================================
二、建议新增的 Vim 快捷键映射
================================================================================

【移动类】

Tab + w        → Option+右箭头     # 按单词向右移动 (word forward)
Tab + b        → Option+左箭头     # 按单词向左移动 (word backward)
Tab + 0        → Cmd+左箭头        # 行首 (与F10冲突，建议用其他键)
Tab + 4 ($)    → Cmd+右箭头        # 行尾 (与F4冲突，考虑用 ; 或 ')
Tab + g        → Cmd+上箭头        # 文件开头 (gg)
Tab + G(shift) → Cmd+下箭头        # 文件结尾

建议映射：
Tab + ;        → Cmd+左箭头        # 行首（分号在右手，方便）
Tab + '        → Cmd+右箭头        # 行尾（引号紧挨分号）
Tab + [        → Cmd+上箭头        # 文件开头
Tab + ]        → Cmd+下箭头        # 文件结尾


【编辑类】

Tab + u        → Cmd+Z             # 撤销 (undo)
Tab + r        → Cmd+Shift+Z      # 重做 (redo)
Tab + x        → Delete            # 删除当前字符
Tab + d        → Cmd+Shift+K      # 删除整行（VS Code/Cursor）
Tab + o        → Cmd+Enter         # 在下方新建一行
Tab + O(shift) → Cmd+Shift+Enter  # 在上方新建一行


【复制粘贴】

Tab + y        → Cmd+C             # 复制 (yank)
Tab + p        → Cmd+V             # 粘贴 (paste)
Tab + Y(shift) → Cmd+X             # 剪切


【搜索】

Tab + /        → Cmd+F             # 搜索
Tab + n        → Cmd+G             # 下一个匹配
Tab + N(shift) → Cmd+Shift+G      # 上一个匹配


【其他实用】

Tab + a        → Cmd+右箭头 然后继续输入  # 在行尾追加（简化为行尾）
Tab + i        → 无需映射，直接输入即可
Tab + e        → Option+右箭头     # 移动到单词末尾（同 w）
Tab + f        → Cmd+D             # 选中当前单词（VS Code 多选）
Tab + s        → Cmd+S             # 保存
Tab + q        → Cmd+W             # 关闭当前标签页

================================================================================
四、与现有配置的冲突处理
================================================================================

冲突点：Tab + 数字键 已映射为 F1~F12

解决方案：
- Vim 的 0（行首）和 $（行尾）用其他键代替
- 建议用 ; 和 ' 代替（右手小拇指区域，顺手）
- 或者用 Home/End 的传统键位 Tab+n/m（如果不用）
