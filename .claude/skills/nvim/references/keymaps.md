# Neovim 快捷键手册

> Leader 键: `<Space>` (空格键)

---

## 📋 基础编辑

| 快捷键 | 模式 | 功能 |
|--------|------|------|
| `S` | N | 保存文件 |
| `Q` | N | 退出 |
| `U` | N | 重做 |
| `;` | N,V | 命令模式 |
| `Y` | V | 复制到系统剪贴板 |
| `` ` `` | N,V | 切换大小写 |
| `<M-j>` | N,V | 连接行 |

---

## 🚀 快速移动

| 快捷键 | 模式 | 功能 |
|--------|------|------|
| `K` | N,V | 向上 15 行 |
| `J` | N,V | 向下 15 行 |
| `W` | N,V | 向前 5 个单词 |
| `B` | N,V | 向后 5 个单词 |
| `H` | N,V,O | 行首 |
| `L` | N,V,O | 行尾 |

---

## 🧭 跳转导航

| 快捷键 | 模式 | 功能 |
|--------|------|------|
| `<C-i>` | N,V | 后退跳转 |
| `<C-o>` | N,V | 前进跳转 |
| `,.` | N,V | 匹配括号 |
| `[c` | N | 跳转到上下文 |
| `\v` | N | 选择到行尾 |

---

## 🪟 窗口管理

| 快捷键 | 功能 |
|--------|------|
| `<leader>sk` | 切换到上方窗口 |
| `<leader>sj` | 切换到下方窗口 |
| `<leader>sh` | 切换到左侧窗口 |
| `<leader>sl` | 切换到右侧窗口 |
| `sk` | 上方分割 |
| `sj` | 下方分割 |
| `sl` | 右侧分割 |
| `srh` | 水平布局转换 |
| `srv` | 垂直布局转换 |
| `<Up>/<Down>` | 调整高度 |
| `<Left>/<Right>` | 调整宽度 |

## 🔍 搜索导航

|--------|------|

| `<C-f>` | FZF 搜索 |

| `<C-h>` | 最近文件 |
