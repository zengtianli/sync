2. 添加 zoxide 和 atuin 初始化
   文件: ~/Documents/sync/zsh/config/modern-tools.zsh 或新建 tools-init.zsh
   
   添加内容:
   ```zsh
   # Zoxide - 智能目录跳转
   if command -v zoxide &> /dev/null; then
     eval "$(zoxide init zsh)"
     alias cd="z"      # 替代 cd
     alias cdi="zi"    # 交互式选择
   fi
   
   # Atuin - Shell 历史管理
   if command -v atuin &> /dev/null; then
     eval "$(atuin init zsh)"
   fi
   ```

3. 添加更多实用别名
   ```zsh
   # 快速编辑核心配置
   alias ez="nvim ~/Documents/sync/zsh/zshrc"
   alias ek="nvim ~/Documents/sync/karabiner/karabiner.json"
   alias ey="nvim ~/Documents/sync/yabai/config/yabairc"
   alias ev="nvim ~/Documents/sync/nvim/init.lua"
   
   # 快速 cd 到常用目录
   alias cdp="cd ~/Downloads/zdwp/02_业务项目"
   alias cds="cd ~/Documents/sync"
   alias cdu="cd ~/useful_scripts"
   alias cdc="cd ~/cursor-shared"
   ```
   所有的alias 都不用，容易和命令 起冲突，这个也记住，到记忆里。除非我要求加入alias，甚至alias的agent，其他就不要多此一举
