   - Loop（已配置）
   - Amethyst（可卸载，与 Yabai 功能重复）

   - CleanMyMac_5
   - Pearcleaner
   - AppCleaner
   建议：保留一个即可，推荐 Pearcleaner（免费轻量）

🔴 【系统监控】有重复
   - iStat Menus（付费，功能全）
   - Stats（免费开源）
   建议：二选一，iStat Menus 更强大

🟡 【AI 客户端】较多
   - ChatGPT、Claude、Doubao、通义、Qwen、腾讯元宝
   建议：按需使用，主力用 Claude/ChatGPT

🟡 【办公套件】有重复
   - Microsoft Office（主力）
   - Apple iWork（轻量）
   - LibreOffice（备用）
   建议：主用 Office，iWork 做简单文档

================================================================================
分析：可能未充分利用的应用
================================================================================

⚠️ 【高价值但可能未充分使用】

1. Hazel - 文件自动整理
   用途：设置规则自动整理下载文件夹、归档旧文件
   建议：配置规则自动分类 Downloads 里的文件

2. Keyboard Maestro - 宏自动化
   用途：文本扩展、触发器、自动化工作流
   建议：创建常用文本片段、自动化重复操作

3. ProFind - 高级搜索
   用途：正则搜索、批量重命名、FTP
   建议：替代 Finder 进行复杂文件操作

4. Screen Studio - 专业录屏
   用途：自动缩放、美化录屏
   建议：做教程视频时使用

5. Easydict - 翻译
   用途：划词翻译、OCR 翻译、支持多引擎
   建议：设置快捷键划词翻译

6. SoundSource - 音频控制
   用途：单独控制每个应用的音量
   建议：会议时降低其他应用音量

7. iStat Menus - 系统监控
   用途：菜单栏显示 CPU/内存/网络/温度
   建议：配置需要监控的指标

8. Mathpix Snipping Tool - 数学公式 OCR
   用途：截图识别数学公式转 LaTeX
   建议：看论文时用
