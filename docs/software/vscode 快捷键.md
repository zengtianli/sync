## VSCode 文件导航快捷键详细表格

### 内置快捷键方案

| 功能分类 | 快捷键 | 说明 |
|:---|:---|:---|
| **切换焦点** |  |  |
| 聚焦到文件浏览器 | `Cmd+Shift+E` | 打开并聚焦到资源管理器侧边栏 |
| 聚焦到侧边栏 | `Cmd+0` | 直接聚焦到当前打开的侧边栏 |
| 返回编辑器 | `Cmd+1` | 聚焦到第一个编辑器组 |
| **文件列表操作** |  |  |
| 向上选择 | `↑` | 在文件列表中向上移动选择 |
| 向下选择 | `↓` | 在文件列表中向下移动选择 |
| 打开文件 | `Enter` | 打开选中的文件 |
| 预览文件 | `Space` | 预览文件内容，不改变编辑器焦点 |
| **快速打开** |  |  |
| 快速打开文件 | `Cmd+P` | 打开快速打开面板，支持模糊搜索 |

### 文件间切换快捷键

| 功能 | 快捷键 | 说明 |
|:---|:---|:---|
| 切换最近文件 | `Ctrl+Tab` | 显示并切换最近打开的文件列表 |
| 切换到上一个标签 | `Cmd+Alt+←` | 在打开的标签页间向左切换 |
| 切换到下一个标签 | `Cmd+Alt+→` | 在打开的标签页间向右切换 |
| 面包屑导航 | `Cmd+Shift+.` | 打开面包屑导航菜单 |
| 当前文件符号 | `Cmd+Shift+O` | 在当前文件中跳转到符号 |
| 工作区符号 | `Cmd+T` | 在整个工作区中搜索符号 |

### 文件操作快捷键

| 操作 | 快捷键 | 说明 |
|:---|:---|:---|
| **删除操作** |  |  |
| 删除文件/文件夹 | `Cmd+Backspace` | 将选中的文件移到回收站 |
| **基础操作** |  |  |
| 重命名 | `Enter` | 在文件浏览器中重命名选中的文件 |
| 复制 | `Cmd+C` | 复制选中的文件或文件夹 |
| 剪切 | `Cmd+X` | 剪切选中的文件或文件夹 |
| 粘贴 | `Cmd+V` | 粘贴文件到当前位置 |
| **快捷键管理** |  |  |
| 打开快捷键设置 | `Cmd+K Cmd+S` | 查看和修改所有快捷键 |
| 打开命令面板 | `Cmd+Shift+P` | 可以搜索并执行任何命令 |

### 自定义快捷键配置示例

| 配置目标 | 建议快捷键 | 命令 | 说明 |
|:---|:---|:---|:---|
| 下一个文件并打开 | `Cmd+]` | `multiCommand.nextFileAndOpen` | 需要 multi-command 扩展 |
| 上一个文件并打开 | `Cmd+[` | `multiCommand.prevFileAndOpen` | 需要 multi-command 扩展 |
| 聚焦并向下选择 | `Cmd+]` | `workbench.action.focusSideBar` + `list.focusDown` | 组合命令实现 |
| 聚焦并向上选择 | `Cmd+[` | `workbench.action.focusSideBar` + `list.focusUp` | 组合命令实现 |

### 扩展插件快捷键

| 扩展名称 | 默认快捷键 | 功能 |
|:---|:---|:---|
| File Navigator | `Alt+Down` | 打开下一个文件 |
| File Navigator | `Alt+Up` | 打开上一个文件 |
| Quick File Navigation | 需自定义 | 快速文件导航功能 |
