# PowerPoint macOS 完整操作指南

---

## 🎯 核心快捷键速查表

| 操作类型 | 快捷键组合 | 功能描述 | 使用场景 |
|----------|------------|----------|----------|
| **格式操作** | `Cmd+Shift+C` → `Cmd+Shift+V` | 复制格式 → 粘贴格式 | 统一多个对象的样式设计 |
| **字体调整** | `Cmd+Shift+>` / `Cmd+]` | 增大字体 | 快速调整文本大小层级 |
| | `Cmd+Shift+<` / `Cmd+[` | 减小字体 | 精确控制文本视觉权重 |
| **对象管理** | `Cmd+Option+G` / `Cmd+Option+Shift+G` | 组合对象 / 取消组合 | 批量管理复杂布局元素 |
| **图层控制** | `Cmd+Shift+]` / `Cmd+Shift+[` | 置于顶层 / 置于底层 | 调整对象显示优先级 |
| **高效复制** | `Cmd+D` | 原地复制并选中新对象 | 比传统复制粘贴更快速 |

## 🖼️ 图像处理与缩放工作流

| 处理类型 | 操作方法 | 精确控制 | 效率技巧 |
|----------|----------|----------|----------|
| **等比缩放** | 选中图片 + `Shift` + 拖动角点 | 保证图像不变形的基线操作 | 避免图像拉伸变形问题 |
| **键盘缩放** | `Cmd+Shift+>` / `Cmd+Shift+<` | 步进式放大/缩小 | 多对象对齐时精确微调 |
| **数值缩放** | `Cmd+Shift+1` 打开格式窗格 | 输入精确像素/百分比值 | 像素级精确控制尺寸 |
| **批量处理** | 选择多个对象 + 统一缩放 | 保持相对比例关系 | 大量图片批量标准化 |

## 📐 对象对齐与布局管理

| 对齐方式 | 菜单路径 | 快捷操作 | 应用场景 |
|----------|----------|----------|----------|
| **相互对齐** | `形状格式 > 排列 > 对齐 > 对齐所选对象` | 先选择参考对象 | 多个元素精确对齐 |
| **页面对齐** | `形状格式 > 排列 > 对齐 > 对齐幻灯片` | 以幻灯片为基准 | 单个对象居中定位 |
| **分布均匀** | `形状格式 > 排列 > 分布` | 等间距自动排列 | 创建整齐的网格布局 |
| **组合居中** | `Cmd+Option+G` 组合 → 整体居中 | 复杂对象统一管理 | 多元素组合精确定位 |

## 🎨 格式刷高级应用技巧

| 使用模式 | 操作方法 | 适用场景 | 效率提升 |
|----------|----------|----------|----------|
| **单次应用** | 选择源对象 → `Cmd+Shift+C` → 选择目标 → `Cmd+Shift+V` | 偶尔使用，简单格式复制 | 减少菜单操作步骤 |
| **连续应用** | 双击格式刷图标 → 依次点击目标对象 | 多个对象需要相同格式 | 避免重复选择源对象 |
| **批量应用** | 先多选目标对象 → 再应用格式 | 大量对象统一格式化 | 一次操作完成批量处理 |
| **取消连续** | 按 `Esc` 键或再次点击格式刷 | 结束连续格式刷模式 | 避免误操作继续应用 |
