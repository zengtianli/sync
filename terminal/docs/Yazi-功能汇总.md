## 技术栈与架构

- **主要语言**: Rust
- **配置语言**: TOML, Lua
- **插件系统**: Lua-based
- **UI框架**: Terminal UI (TUI)

## 核心功能模块

### 1. 文件管理核心

- **文件浏览与导航**
  - 多面板布局 (比例可配置: 1:3:4)
  - 支持隐藏文件显示切换
  - 符号链接显示支持
  - 滚动偏移量配置 (默认5行)

- **文件操作**
  - 复制/剪切/粘贴
  - 重命名 (支持多种模式)
  - 创建文件/目录
  - 删除 (支持回收站)
  - 硬链接/软链接创建

### 2. 高级浏览功能

- **排序系统**
  | 排序方式 | 支持方向 |
  |----------|----------|
  | 字母顺序 | 正序/逆序 |
  | 修改时间 | 正序/逆序 |
  | 创建时间 | 正序/逆序 |
  | 文件大小 | 正序/逆序 |
  | 扩展名 | 正序/逆序 |
  | 自然排序 | 正序/逆序 |

- **搜索与过滤**
  - 实时文件过滤
  - 内容搜索 (ripgrep集成)
  - 智能查找
  - FZF集成
  - Zoxide集成

### 3. 预览系统

- **文件预览**
  - 代码高亮
  - 图片预览
    - 支持多种图片格式
    - 可配置图片质量和过滤器
  - 视频预览
  - PDF预览
  - 字体预览
  - 压缩包预览

### 4. 任务管理

- 后台任务系统
  - 微任务工作者: 10
  - 宏任务工作者: 25
  - 异常重试机制
  - 图像内存分配控制
