# Yazi 文件管理器 - 功能与快捷键指南

## 简介

Yazi 是一个快速、功能丰富的终端文件管理器，使用 Rust 编写。本指南总结了配置中定义的主要功能和快捷键。

## 核心功能

- 文件浏览与导航
- 书签管理
- 文件压缩
- 文件搜索与过滤
- 多标签支持
- Git 集成
- 视觉模式选择
- 文件操作（复制、剪切、粘贴、重命名、删除）

## 常用快捷键

### 基础操作

| 快捷键 | 功能 |
|--------|------|
| `<Esc>`, `<C-[>` | 退出视觉模式、清除选择或取消搜索 |
| `q` | 关闭当前标签页，如果是最后一个则退出 |
| `Q` | 退出程序（不写入 cwd 文件） |
| `R` | 刷新当前目录 |

### 导航

| 快捷键 | 功能 |
|--------|------|
| `j`, `<Down>` | 向下移动光标 |
| `k`, `<Up>` | 向上移动光标 |
| `h`, `<Left>` | 返回父目录 |
| `l`, `<Right>`, `<Enter>` | 进入子目录或打开文件（使用 smart-enter 插件） |
| `H` | 返回上一个目录 |
| `L` | 前进到下一个目录 |
| `<C-u>` | 向上移动半页 |
| `<C-d>` | 向下移动半页 |
| `gg` | 移动到顶部 |
| `G` | 移动到底部 |

### 文件选择

| 快捷键 | 功能 |
|--------|------|
| `<Space>` | 切换当前选择状态并向下移动 |
| `v` | 进入视觉模式（选择模式） |
| `V` | 进入视觉模式（取消选择模式） |

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `yy` | 复制选中的文件 |
| `dd` | 剪切选中的文件 |
| `dD` | 永久删除选中的文件 |
| `pp` | 粘贴文件 |
| `pP` | 强制粘贴文件（覆盖已存在的目标） |
| `pl` | 创建绝对路径符号链接 |
| `pL` | 创建相对路径符号链接 |
| `ph` | 创建硬链接 |

### 重命名

| 快捷键 | 功能 |
|--------|------|
| `cw` | 重命名选中的文件 |
| `a` | 在扩展名前重命名 |
| `A` | 在文件名末尾重命名 |

### 创建文件/目录

| 快捷键 | 功能 |
|--------|------|
| `T` | 创建文件（以 / 结尾表示目录） |
| `M` | 创建目录 |

### 搜索与过滤

| 快捷键 | 功能 |
|--------|------|
| `<C-p>` | 使用 fzf 跳转到目录或显示文件 |
| `zo` | 使用 zoxide 跳转到目录 |
| `F` | 使用 ripgrep 按内容搜索文件 |
| `f` | 过滤文件 |
| `/` | 智能查找下一个文件 |
| `?` | 智能查找上一个文件 |
| `n` | 跳转到下一个匹配项 |
| `N` | 跳转到上一个匹配项 |

### 书签功能

| 快捷键 | 功能 |
|--------|------|
| `'a` | 添加书签 |
| `''` | 使用 fzf 跳转到书签 |
| `'r` | 按键删除书签 |

### 压缩功能

| 快捷键 | 功能 |
|--------|------|
| `ca` | 压缩选中的文件 |

### Shell 操作

| 快捷键 | 功能 |
|--------|------|
| `<C-g>` | 启动 Lazygit |
| `S` | 启动 shell（带确认） |
| `;` | 运行交互式 shell 命令 |
| `:` | 运行阻塞式交互 shell 命令 |

| 快捷键 | 功能 |
|--------|------|

| `mp` | 设置行模式为权限 |

### 标签页管理

| 快捷键 | 功能 |
|--------|------|
| `tt` | 用当前工作目录创建新标签页 |
| `th` | 切换到上一个标签页 |

### 其他功能

| 快捷键 | 功能 |
|--------|------|
| `zh` | 切换隐藏文件的可见性 |
| `r` | 交互式打开选中的文件 |

## 特殊模式快捷键

配置还包含了其他模式的快捷键，如：

- Spot 模式 - 用于浏览和搜索
- Tasks 模式 - 用于管理后台任务
- Pick 模式 - 用于选择
- Input 模式 - 用于输入
- Completion 模式 - 用于自动完成
- Help 模式 - 用于查看帮助

每个模式都有其特定的快捷键集，以支持其特定功能。

## 插件

该配置使用了多个插件：

- `smart-enter` - 智能进入目录或打开文件
- `yamb` - 书签管理
- `compress` - 文件压缩
- `fzf` - 模糊查找
- `zoxide` - 智能目录跳转

要充分利用这些功能，确保相关插件已正确安装并配置。
