四、建议操作
================================================================================

【方案A：快捷键调用脚本（简单快速）】

在 keymap.toml 的 [mgr] prepend_keymap 中添加：

```toml
# 文档转换快捷键
{ on = ["c", "d", "m"], run = 'shell "bash ~/useful_scripts/execute/raycast/docx_to_md.sh $@" --confirm', desc = "DOCX → MD" },
{ on = ["c", "m", "d"], run = 'shell "bash ~/useful_scripts/execute/raycast/md_to_docx.sh $@" --confirm', desc = "MD → DOCX" },
{ on = ["c", "x", "c"], run = 'shell "python3 ~/useful_scripts/execute/raycast/xlsx_to_csv.py $@" --confirm', desc = "XLSX → CSV" },
{ on = ["c", "c", "x"], run = 'shell "python3 ~/useful_scripts/execute/raycast/csv_to_xlsx.py $@" --confirm', desc = "CSV → XLSX" },

# DOCX 格式化
{ on = ["c", "f", "a"], run = 'shell "python3 ~/useful_scripts/execute/raycast/docx_zdwp_all.py $@" --confirm', desc = "DOCX ZDWP格式化" },
{ on = ["c", "f", "t"], run = 'shell "python3 ~/useful_scripts/execute/raycast/docx_text_formatter.py $@" --confirm', desc = "DOCX 文本修复" },

# 读取预览
{ on = ["c", "r", "d"], run = 'shell "python3 ~/useful_scripts/execute/tools/read_docx.py $@ | less" --block', desc = "读取 DOCX" },
{ on = ["c", "r", "x"], run = 'shell "python3 ~/useful_scripts/execute/tools/read_xlsx.py $@ | less" --block', desc = "读取 XLSX" },

# 文件比较
{ on = ["c", "c", "f"], run = 'shell "python3 ~/useful_scripts/execute/compare/compare_files_folders.py $@" --block', desc = "文件比较" },
```

五、快捷键规划建议
================================================================================

按照你的 keymap 风格，建议使用 "c" 前缀作为"转换/命令"系列：

c + 第二键（类型）+ 第三键（目标）

【转换类快捷键】
  cdm = DOCX → Markdown
  cmd = Markdown → DOCX
  cdp = DOCX → PDF
  cpm = PDF → Markdown

  cxc = XLSX → CSV
  ccx = CSV → XLSX
  cxt = XLSX → TXT

  cpm = PPTX → Markdown

【格式化快捷键】
  cfa = DOCX 全部格式化 (ZDWP All)
  cft = DOCX 文本修复
  cfh = DOCX 标题样式
  cfp = DOCX 段落样式
  cfs = DOCX 表格样式

【读取/预览】
  crd = 读取 DOCX
  crx = 读取 XLSX
  crp = 读取 PDF

【比较】
  ccf = 比较文件
  ccx = 比较 Excel

【Markdown】
  cmf = Markdown 格式修复
  cmm = Markdown 合并
  cms = Markdown 拆分
